Rules

Mandatory rules

- You're not allowed to create any extra tag in HTML or javascript. Use only the ones available inside the USEFUL MARKUP comments.

- Keep your css and javascript code organized for a good readability

- You'll create a crossing. There are 2 one-way streets. The traffic moves from left to right in horizontal road and from top to bottom in vertical road. There are 4 blocks, and the viewport is limited by those 4 blocks. The cars shouldn't appear outside this viewport.

- You should reproduce the CSS as most as similar to the image called reference.png attached in this folder. Do not use background images in this exercise.

- Traffic lights rules: The lights should toggle every 15 seconds (both must be red during 3 seconds before toggle the lights). A car should never pass trough a red light in his road.

Recommendations

- Every car should move between 50px and 92px per second.

- The cars should move smooth.

- Try use javascript without any framework.
